1、工具

Node.js

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。Node.js 的包管理器 npm，是全球最大的开源库生态系统。

Node.js官网，官网下载安装好Node。

有版本输出即表示安装成功。

安装完Node之后，推荐装一下npm的淘宝镜像，后面npm install的时候速度会快很多，也会更稳定。

Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

易用，部署简单，基于Node.js，速度确实很快。

Hexo官网，文档，API，插件等等官网上都有。

Git

Git就不多介绍了，你需要一个Git账号，掌握基本的clone,push和pull操作即可。新用户注意在github上添加ssh_key。

后续

第三方服务

现在很多成熟的主题一般都会集成一些第三方服务，根据主题的配置文件，完成相应的填充和不全即可。

比较推荐的第三方有下面几个：

  • 多说评论，支持自定义，相较于Disqus，支持微博等国内社交平台的登录方式。
  • Swiftpye搜索，有30天试用期，30天过后依旧有免费版可用。支持自定义搜索样式。
  • jiathis分享，日志底部的分享按钮。
  • 百度统计，按个人喜好添加。

域名

完成上述配置后，只能通过githubID.github.io来访问自己的博客。

如果需要定制个人域名，可以去狗爹或者阿里云购买域名。

然后在git仓库的根目录下添加一个CNAME文件，文件中只需写上你申请的域名。

最后在你的域名配置中，指向githubID.github.io即可。

同步

推荐将my_blog目录也push到git上去，这样就不用担心hexo静态文件的丢失了。
